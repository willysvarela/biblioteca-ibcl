## Biblioteca IBCL
Web App que lista os livros disponíveis para aluguel e reserva na Biblioteca da IBCL.

## Como rodar localmente:

Rodar no terminal:

```bash
npm run dev
# ou
yarn dev
```

O app ficará disponível em [http://localhost:3000](http://localhost:3000).
